# AttackBarbarians
这是一个肉鸽类游戏 - 单机游戏类型，主要是为游戏玩家提供爽快的游戏体验

游戏核心玩法是：player 在一个无限模式的游戏场景中杀死敌人，获取游戏技能升级和游戏装备提升，该游戏默认为无限模式，随着游戏时间增长，战场获得提升效果，战场会为enemy随机提升某一项属性，或者随机生成boss，或者两者都有，enemy本身也会随着战场时间提升自身的属性

当enemy击破战场靠近player的防护墙之后，本次对战结束，结算本次对战中玩家回去的技能碎片和装备（技能碎片用于提升技能等级，装备用于合成更高等级的装备）
