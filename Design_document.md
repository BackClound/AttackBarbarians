## 1. 产品概述

### 1.1 一句话定位

玩家在 **无限生存模式** 场景中持续清怪，随时间敌人全面变强；通过 **局内经验升级 → 周期性基础 Buff** 与 **里程碑技能强化** 获得爽感；通过 **局外技能等级 / 基础四维等级 / 货币** 形成「打不过 → 多局积累 → 打更久」的长期生命周期。

### 1.2 对标与体验方向（玩法层面）

- **《向僵尸开炮》类**：波次压力、技能构筑、高密度战斗反馈、局内三选一成长。  
- **《佣兵的战争》类**：局外成长、属性与技能长期积累、可见战力提升。

> 不涉及具体 IP 资产复刻，仅借鉴体验结构。

### 1.3 核心体验关键词

爽快反馈 · 构筑可见成长 · 10 分钟休闲门槛 · 局外兜底 · 经济克制

### 1.4 目标用户与难度基调

- **休闲玩家** 约在 **10 分钟** 起进入明显高压区；通过多局积累升级卡逐步延长存活时间。  
- 期望中位休闲玩家在合理局外进度下，**第 5～15 局** 能较稳定触及或跨越 10 分钟压力带（需实测调表）。

### 1.5 分期范围

| 阶段 | 内容 |
| --- | --- |
| **一期** | 无限模式、敌人随时间与种类强化、Boss 时间点、局内经验与升级、每 4 次升级基础 Buff、里程碑技能 Buff、技能与基础属性养成、体力、商城、钻石多来源、抽奖、签到、离线巡逻、广告礼包、普通/精英模式 |
| **二期** | **装备系统**（一期商城不展示或灰置装备入口） |

## 2. 核心玩法循环

### 2.1 单局循环（Session Loop）

```
消耗体力 → 进入战斗 → 清怪/走位/释放技能
    → 经验（时间 + 击杀）→ 升级事件
    → 每累计 4 次升级事件 → 基础属性 Buff 三选一
    → 时间推进：敌人变强、敌种池变化、刷怪加压
    → 5/10/15/20 分钟：精英 Boss + 技能向强化（里程碑）
    → 死亡/结束 → 结算（整卡为主）→ 养成界面
```

### 2.2 长期循环（Meta Loop）

```
累计局内时长解锁技能
    → 技能升级卡提升技能等级
    → 基础属性升级卡提升局外四维
    → 单局存活更久 → 更高时长档位奖励
    → 金币/钻石/券 → 商店与抽奖加速
    → 体力与时间节奏控制每日游玩量
```

### 2.3 设计原则（评审用）

1. **爽感与可读性**：同屏敌人上限；移速/攻速软上限，避免弹幕不可读。  
2. **局内不碾压局外**：局内 Buff 递减 + 敌人成长曲线，避免一局定终身。  
3. **局外兜底**：多局整卡 + 时长解锁，休闲玩家能逐步突破 10 分钟带。  
4. **经济克制**：钻石多来源但频控清晰；抽奖公示与保底。

## 3. 模式设计

### 3.1 普通模式（Normal）

- 敌人四维成长与结算为 **基准倍率**。  
- 适合新手、日常清体力、熟悉 Buff 与技能节奏。

### 3.2 精英模式（Elite）

- 敌人 **HP / ATK / 移速 / 攻速** 在普通模式同时间点基础上 **整体略高**（分项可配）。  
- **结算奖励倍率** 高于普通（建议 `elite_reward_beta` 约 0.2～0.5）。  
- 移速/攻速类倍率仍设 **软上限**，保持休闲可反应性。

---

## 4. 关卡与时间轴（无限模式）

### 4.1 时间压力分段（建议）

| 阶段 | 时间 | 设计意图 |
| --- | --- | --- |
| 热身段 | 0～3 分钟 | 密度与属性小幅上升，熟悉操作 |
| 构筑段 | 3～10 分钟 | 更多敌种组合，开始需要 Buff 构筑 |
| 高压段 | 10 分钟+ | 属性斜率提高，休闲门槛主战场 |

### 4.2 精英 Boss 里程碑

- **5 / 10 / 15 / 20 分钟**（可扩展）生成精英 Boss。  
- Boss 需有 **机制差异**（预警范围、阶段、召唤等），避免纯血牛。  
- 击杀给予：额外局内收益（经验/临时 Buff）+ **更高结算权重**。

### 4.3 里程碑技能向 Buff

- 与 Boss 或时间点绑定，强化 **技能效果维度**（范围、穿透、持续、控制等）。  
- 与「每 4 级一次的基础属性 Buff」分工：**前者改构筑形态，后者抬数值底盘**。

### 4.4 敌人种类随机池

- 随时间解锁更高威胁敌种；权重表 + 「近期不重复」规则，减少坏随机。

## 5. 敌人成长机制（局内）

### 5.1 四维属性

随局内时间 \(t\)（分钟）增长：

- **血量（HP）**  
- **攻击力（ATK）**  
- **移动速度（MS）** — 有硬上限  
- **攻击频率/攻速（AS）** — 有硬上限  

精英模式在普通结果上再乘 \(e_{hp}, e_{atk}, e_{ms}, e_{as}\)（整体略升，移速/攻速更克制）。

### 5.2 数值公式（策划/程序表驱动）

对每种敌人基础四维 \(HP_0, ATK_0, MS_0, AS_0\)：

\[
M_{hp}(t)=1 + a_{hp}\cdot t^{p_{hp}},\quad
M_{atk}(t)=1 + a_{atk}\cdot t^{p_{atk}}
\]

\[
M_{ms}(t)=\mathrm{clamp}(1 + a_{ms}\cdot t^{p_{ms}},\ 1,\ M^{max}_{ms})
\]

\[
M_{as}(t)=\mathrm{clamp}(1 + a_{as}\cdot t^{p_{as}},\ 1,\ M^{max}_{as})
\]

普通模式：

\[
HP(t)=HP_0\cdot M_{hp}(t),\ \ldots
\]

精英模式：

\[
HP^E(t)=HP(t)\cdot e_{hp},\quad ATK^E(t)=ATK(t)\cdot e_{atk},\ \ldots
\]

**建议默认区间**：\(e_{hp}, e_{atk} \in [1.08, 1.18]\)；\(e_{ms}, e_{as} \in [1.03, 1.10]\)。

**联调占位**：`a_hp=0.06,p_hp=1.35; a_atk=0.05,p_atk=1.25; M_ms^max=1.45; M_as^max=1.35; e_hp=1.12,e_atk=1.10,e_ms=1.05,e_as=1.05`

## 6. 局内成长：经验、升级事件与基础 Buff

### 6.1 经验条

| 来源 | 说明 |
| --- | --- |
| 时间项 | 随局内时间持续获得 \(\frac{dExp}{dt}=B_{time}\cdot D\) |
| 击杀项 | 每击杀 \(\sum K_i \cdot w_i \cdot D\)（敌种可配权重 \(w\)） |

\(D\) 为模式系数（普通=1，精英可微调）。两者汇入 **同一经验条**；满则升级，溢出保留。

### 6.2 升级所需经验

\[
NeedExp(L)=E_0\cdot L^{g}\cdot \exp(\lambda\cdot \max(0,L-L_0))
\]

- 前期升级快，便于稳定触发「每 4 级 Buff」。  
- 后期 `exp` 项放缓，避免局内无限碾压。  

**10 分钟休闲目标**：多数玩家在 10 分钟附近达到 \(L \in [6, 10]\)。

**联调占位**：`E0=80, g=1.35, λ=0.02, L0=8, B_time=6, K=12~30`（按敌种）

### 6.3 升级事件与基础 Buff 触发

- 每次升级：`buffEventCounter += 1`。  
- 累计 **4 次**：弹出 **基础属性 Buff 三选一**（攻击/攻速/移速/生命等），选择后 `counter -= 4`。  
- **中断不重置**：暂停、切后台不清计数。  
- **跨局是否累计**：由 `buff_event_counter_persist` 配置（默认建议仅局内）。

### 6.4 基础 Buff 叠层递减

对同一属性第 \(n\) 次选择（本局）：

\[
\Delta X_n = \Delta X_{base}\cdot r^{\,n-1},\quad 0<r<1
\]

**联调占位**：`r=0.78`；\(\Delta X_{base}\) 按属性分表。

## 7. 技能系统

### 7.1 技能列表（一期）

| 技能 | 设计侧重 |
| --- | --- |
| 基础射击技能 | 普攻主轴；射速、弹道数、穿透、伤害 |
| 闪电攻击技能 | 链式/弹射；链接数、衰减、频率 |
| 落雷攻击技能 | 区域雷击；范围、预警、硬直 |
| 火雨技能 | 持续 AOE；持续时间、密度、灼烧叠层 |
| 水波纹减速攻击技能 | 控制；减速强度、范围、时长（有上限） |
| 火环禁锢技能 | 强控；禁锢时长、半径、冷却（精英可有抗性） |

### 7.2 升级卡规则

- **专属技能升级卡**：绑定 `skillId`。  
- **通用技能升级卡**：**完全替代** 专属卡；任意比例凑满需求张数。  
- 升级消耗随等级递增（见 §8.2）。

### 7.3 技能解锁主路径（冻结）

以 **累计局内游戏时长** `total_play_seconds`（可排除暂停）为主路径，示例门槛（可改表）：

| 累计时长 | 解锁技能 |
| --- | --- |
| 默认 | 基础射击 |
| ≥ 10 分钟 | 闪电 |
| ≥ 30 分钟 | 落雷 |
| ≥ 60 分钟 | 火雨 |
| ≥ 120 分钟 | 水波纹减速 |
| ≥ 180 分钟 | 火环禁锢 |

签到/抽奖等 **不作为主路径**，避免玩家认知混乱。

### 7.4 局外基础四维

- 攻击、生命、移速、攻速（或项目定义四维）独立升级。  
- 每级消耗 **基础属性升级卡** 数量递增；影响单局初始或全局系数（乘算/加算在 PRD 与实现中统一）。

## 8. 局外养成与消耗曲线

### 8.1 基础四维升级消耗（整卡）

\[
N_{attr}(L_a)=\left\lceil A_0 + A_1\cdot L_a + A_2\cdot L_a^{2} \right\rceil
\]

### 8.2 技能等级升级消耗（整卡）

\[
N_{skill}(L_s)=\left\lceil S_0 + S_1\cdot L_s + S_2\cdot L_s^{2} \right\rceil
\]

专属卡、通用卡或混合，均需凑满 \(N_{skill}\) 张。

**联调占位**：`A0=2,A1=1,A2=0.06; S0=2,S1=1.2,S2=0.08`

### 8.3 生命周期设计意图

- 早期线性项主导 → 快速感知成长。  
- 中后期二次项 → 拉长周目，靠结算整卡与玩法产出仍可推进。  
- 目标：多打 **3～8 局** 可明显多存活 **30～90 秒**（可量化验收）。

## 9. 结算与奖励

### 9.1 结算输入

局内存活时长、模式（普通/精英）、Boss 击杀、击杀数、里程碑达成等。

### 9.2 结算输出（一期）

- **主奖励**：整卡（技能升级卡、基础属性升级卡）。  
- **辅奖励**：金币、抽奖券、少量钻石（玩法产出）。  
- **时长档位**：越高越好；精英同档倍率更高。  
- **收益上限**：每日或单局封顶，防刷。

### 9.3 结算倍率（精英）

\[
Reward = BaseReward(time)\cdot (1+\beta\cdot \mathbb{I}_{elite})
\]

\(\beta\) 建议 0.2～0.5。

### 9.4 时长档位示例（可配置）

| 存活时长 | 奖励倾向 |
| --- | --- |
| 0～3 分钟 | 保底金币/低级卡 |
| 3～6 分钟 | 基础属性卡 |
| 6～10 分钟 | 更多属性卡 + 低概率技能卡 |
| 10～15 分钟 | 技能卡概率提升 |
| 15～20 分钟 | 高收益区（精英更明显） |
| 20+ 分钟 | 收益递减或日上限 |

## 10. 经济系统

### 10.1 货币定位

| 货币 | 用途 | 主要来源 |
| --- | --- | --- |
| **金币** | 基础属性升级卡、低级商品 | 结算、离线、战斗 |
| **钻石** | 技能卡、抽奖、高级商品 | 广告、玩法产出、商店免费领取 |
| **抽奖券** | 抽奖 | 结算、签到、Boss、商城 |

### 10.2 钻石规则（冻结）

- **广告**：主要付费外获取途径之一（需日上限与失败策略）。  
- **玩法产出**：结算/里程碑等可追踪来源。  
- **商店免费领取**：冷却 \(T_{cool}=12\text{h}\)，每次 \(d=5\) 钻；`now - last\_claim \geq T_{cool}` 才可领。

### 10.3 商城

- 金币店 / 钻石店分池；日/周限购；价格曲线。  
- 免费领取钻石独立入口 + 倒计时。

### 10.4 体力

- 每次开局消耗固定体力；为 0 不可开局。  
- 随时间缓慢恢复；可配置上限与恢复速率。

### 10.5 抽奖

- **不同奖品** 配置 **不同权重或概率**；界面 **公示**。  
- 支持 **钻石直购**；建议保底（\(N\) 抽无稀有则强制稀有一次）。

权重归一化：\(P_i = W_i / \sum_j W_j\)

### 10.6 签到

- **第 1～6 天**：属性升级卡（品质/数量递增）。  
- **第 7 天**：技能向奖励（通用技能卡 + 专属卡等）。  
- 可选：钻石补签（限次）。

### 10.7 随机广告礼包

- 观看广告随机获得升级卡或资源。  
- **每日次数上限**；期望价值 **低于** 同等活跃正常游玩。

### 10.8 离线巡逻

- 按离线时长 **分档** 领取。  
- **最多累计 8 小时**（超出不计）。  
- 奖励挂钩养成但不宜碾压主玩法。

## 11. 调参工作流（休闲 10 分钟门槛）

1. 定 10 分钟目标等级 \(L_{10} \in [6,10]\)。  
2. 反推 \(B_{time}, K, w\) 与 \(NeedExp(L)\)。  
3. 敌人曲线保证 10 分钟 TTK 上升、受伤频率上升。  
4. Buff 递减 + 刷怪密度，防止玩家远超敌人。  
5. 局外 \(N_{attr}, N_{skill}\) 保证多局可见时长延长。
