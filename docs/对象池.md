# 对象池系统规范

## 架构设计
- `PoolManager` 统一管理所有对象池，支持 `Get<T>(string poolKey)` 和 `Return(GameObject obj)`。
- 每个池实例使用 `ObjectPool<T>` 泛型类实现，其中 T 为 MonoBehaviour 类型。
- 池内对象通过 `IPoolable` 接口实现 `OnSpawn()` 和 `OnDespawn()` 生命周期回调，在获取和释放时自动调用。

## 接口定义
```csharp
public interface IPoolable
{
    void OnSpawn();    // 从池中取出时调用
    void OnDespawn();  // 放回池中时调用
}
